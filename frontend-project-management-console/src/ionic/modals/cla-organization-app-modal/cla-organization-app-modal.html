Connect LF CLA App to GitHub Organization

How to install the GitHub App

1. Click on "Install The GitHub CLA App" Below. You will be directed to your organization's GitHub page.

2. Click the "Install" button, located on the top-right corner of your organization's GitHub page.

3. Select the Organization that you have added to the Project Management Console.

4. Select the repositories you want to configure the CLA Application for your Organization.

You have successfully configured CLA for your GitHub Organization. In the Project Management Console, Check that the repositories selected in Step 4 are listed under your GitHub Organization.

If you are not able to see your selected repositories, Please try again from Step 1.
