Connect LF CLA App to GitHub Organization

How to install the GitHub App

1. Click on "Install The GitHub CLA App" Below. You will be directed to your organization's GitHub page.

2. Click the "Install" button, located on the top-right corner of your organization's GitHub page.

3. Select the Organization that you have added to the Project Management Console.

You have successfully connected CLA for your GitHub Organization.

Please use the "Connect Repositories" button in your CLA group below to configure your desired Repositories. You Must configure a repository to a CLA Group in the Project Management Console. Having only the GitHub App installed will not trigger the CLA system.

If you are not a GitHub Administrator, please share these steps with a GitHub Administrator who has permission to install a GitHub application.
